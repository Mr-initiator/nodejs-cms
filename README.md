基于nodeJS MongoDB的网站内容管理系统
===================


管理员界面使用angularjs，单页面应用响应速度快，前后端分离，后端使用express框架，mongoDB数据库。

1.cd nodeJSCMS

2.cd admin

3.npm install

4.npm start

5.cd server

6.npm install

7.npm start

----------

管理员界面默认监听8090端口 http://localhost:8090/#!/login

第一次使用后端会自动生成登录账户和密码 分别为admin admin
